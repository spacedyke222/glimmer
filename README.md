# glimmer
An iOS application built with SwiftUI designed to help diabetic athletes track,visualize, and manage blood glucose trends across workouts.

## ✨ Features

- Real-time data tracking and visualization
- Interactive charts for viewing trends over time
- Clean SwiftUI-based architecture
- Modular structure using Models, Views, and ViewModels
- Lightweight and responsive UI

## 📊 Tech Stack

- Swift
- SwiftUI
- Xcode
- Charts framework
